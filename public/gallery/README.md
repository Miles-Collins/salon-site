# Gallery Images

## How to Add Gatsby Studios Images

1. Go to Google and search for "Gatsby Studios Leavenworth Kansas"
2. Visit their Google Business profile
3. Download the images you want to display
4. Save them in this folder (`public/gallery/`) with names like:
   - `image1.jpg`
   - `image2.jpg`
   - `image3.jpg`
   - etc.

5. Update `src/app/gallery/page.tsx` to match the number and names of images you've added

## Image Requirements
- Format: JPG or PNG
- Recommended size: At least 800x800px for best quality
- The page displays images in a responsive grid (3 columns on desktop, 2 on tablet, 1 on mobile)
